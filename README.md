### Mapping
| Shortcut | Function |
|---|---|
| `s-<return>` | Switch to full-screen mode |
| `C-x o` | **Ace-window**: Jump to the specified window (enter letters) |
| `C-x w` | **Ace-swap**: Swap with target window |
| `S-↑/↓/←/→` | Move window focus by direction |
| `C-x 0` | Close current window (native) |
| `C-x 1` | Close all other windows (native) |
| `C-x C-f` | **Smart find file**: Search in project / Normal open outside project |
| `C-c p` | **Projectile**: Project management command prefix |
| `DEL` | **Vertico**: Delete one directory level in path search |
| `M-x` | Enhanced command palette (with annotations and filtering) |
| `C-x g` | **Magit**: Open Git status panel |
| `C-x C-g` | Magit Dispatch (quickly execute Git commands) |
| `` C-` `` | **Vterm**: Toggle bottom popup terminal |
| `C-c d f` | **Ediff**: Compare two files |
| `C-c d b` | **Ediff**: Compare two buffers |
| `C-c d d` | **Ediff**: Compare two directories |
| `M-.` | Jump to definition |
| `M-,` | Jump back to previous position |
| `C-c C-r` | **Rename**: Rename variable (LSP) |
| `C-c C-f` | **Format**: Force code formatting |
| *(Auto)* | **Format on Save**: Auto format on save |
| `C-x h f` | **Helpful**: View detailed function/macro docs |
| `C-x h v` | **Helpful**: View detailed variable docs |
| `C-x h k` | **Helpful**: View detailed key binding docs |
| `C-x u` | **Undo-Tree**: Enable visual undo tree |
| `C-=` | Text scale increase |
| `C--` | Text scale decrease |
| `<f5>` | Quickly open `init.el` config file |
